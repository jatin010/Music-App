# 🎵 Music App Clone

A sleek, fully functional, and highly responsive music streaming web application built using modern frontend technologies. This application uses the YouTube Iframe Player API to seamlessly stream music tracks embedded inside a gorgeous, glassmorphism-inspired user interface.

## 🚀 Live Demo & Deployment
This project is built purely with static files (`HTML`, `CSS`, and `JavaScript`), making it 100% compatible with **GitHub Pages**, Vercel, Netlify, or any other static hosting platform.

---

## ✨ Features
- **Fluid Audio Streaming:** Powered by the official YouTube Iframe API for stable background streaming.
- **Glassmorphism Interface:** A modern, visual design utilizing backdrop blurs, dark accents, and subtle borders.
- **Custom Player Controls:** Take full control of your playback with intuitive Play/Pause, Skip Forward, Skip Backward, Volume Slider, and an interactive seeking progress bar.
- **Live Search & Filtering:** Instantly filter artists, tracks, or genres in real-time as you type.
- **PWA Ready:** Includes a `manifest.json` file configuring the app for PWA installation, allowing users to add it to their home screen on mobile and desktop devices.
- **Fully Responsive:** Meticulously engineered breakpoint system adjusting dynamically across desktop, tablet, and ultra-compact mobile layouts.

---

## 📂 File Architecture
- `index.html` — The structured layout containing the navbar, interactive search capsule, dynamic playlist display grid, custom music player wrapper, and footer.
- `style.css` — Centralized design engine managing themes, layout grids, animations, fluid transitions, and media-queries.				
- `library.js` — Core application controller overseeing programmatic library injections, event bindings, notification animations, and the YouTube API integration bridge.
- `manifest.json` — Metadata definitions controlling localized settings, standalone orientation flags, launch URL paths, and app shortcuts.
